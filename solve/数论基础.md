#### 欧拉函数

> **欧拉函数** $\varphi \left ( x \right ) $ 是小于等于 $x$ 的正整数中与 $x$ 互质的数的数量，



> 例如 $\varphi \left ( 8 \right ) $ = $4$ ，在小于等于 $8$ 的正整数中，只有 $1，3，5，7$ 符合条件 



其实对于**欧拉函数**，还可以从**简化剩余系**的角度来看。
